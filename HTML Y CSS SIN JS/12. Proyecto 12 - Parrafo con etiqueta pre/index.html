Parrafo con etiqueta pre


            Este es un parrafo con la etiqueta pre, que sirve para mostrar texto tal cual se escribe en el codigo fuente.
            Por ejemplo,     mucho espacio                           si escribimos un texto con saltos de linea, estos se respetaran en la visualizacion del navegador.
            Esto es util para mostrar codigo fuente, o cualquier texto que necesite mantener su formato original.
             La etiqueta bdo con el dir rtl muestra el texto invertido
             ABBR en HTML es un lenguaje de marcado que se utiliza para el desarrollo de paginas web.
